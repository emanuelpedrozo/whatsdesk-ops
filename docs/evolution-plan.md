# Plano de Evolucao (Fase 2+)

## IA Assistiva (com LGPD)
- Sugestao de resposta (human-in-the-loop).
- Resumo de conversa para handoff entre atendentes.
- Classificacao de lead quente/morno/frio.
- Extracao de intencao (preco, prazo, entrega, reclamacao).

## Regras de Governanca
- Nao treinar modelos com dados sensiveis sem consentimento explicito.
- Redacao de PII antes de enviar para modelos externos.
- Trilhas de auditoria para toda acao de IA sugerida/aceita.

## Expansao de Integracoes
- Multiplo BSP via adapters.
- Gateway de pagamentos (Pix/link/cartao).
- ERP/logistica.
